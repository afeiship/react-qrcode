## installation
```shell
npm install -S @jswork/react-qrcode
```

## properties

## usage
1. import css
  ```scss
  @import "~@jswork/react-qrcode/dist/style.css";

  // or use sass
  @import "~@jswork/react-qrcode/dist/style.scss";

  // customize your styles:
  $react-qrcode-options: ()
  ```
2. import js
  ```js
__GENERATE_DAPP__
  ```
